# Rato
RATO (Ratings on OTT) is a Chrome extension that shows IMDb & Rotten Tomatoes ratings directly on OTT platforms like Netflix, right where you’re watching.
No tab switching. No guesswork. Just better decisions.

---Features

-> Displays IMDb ratings directly on OTT platforms
-> Shows Rotten Tomatoes scores (when available)
-> Real-time detection while browsing
-> Zero user input — fully automatic
-> Clean, non-intrusive UI overlay

---Tech Stack

->HTML – UI structure
->CSS – Styling & layout
->JavaScript – Core logic & DOM manipulation
->Chrome Extensions API
->Public Movie & TV APIs
